Access point

  • https://api.quantrade.co.uk/

Endpoints

  • Brokers: /brokers/
  • Symbols: /symbols/
  • Periods: /periods/
  • Trading strategies: /systems/
  • Strategy signals: /<broker_slug>/<symbol>/<period_minutes>/<strategy>/<api_key>/
  • Stats:: /stats/<broker_slug>/<symbol>/<period_minutes>/<strategy>/<trade_direction>/
  • Index components:: /indexes/<index>/<broker_slug>/components/
  • Latest index signals: /indexes/<index>/<broker_slug>/latest/<days>/
  • Index results: /indexes/<index>/<broker_slug>/results

Formats, methods

  • json, get

Example

First you need to know broker slug from list of brokers, symbols, period and strategy from lists of respective entities and then request signals.

  • /ava_trade_eu_ltd/GOOGLE/1440/WND1/

Limits

No limits currently whatsoever, except that signals are delayed by 1 bar (day, wek or month) for non-authenticated users (don't use <api_key> or enter anything instead).

Authentication

Authentication is needed only for strategies (i.e. signals) endpoint.

API key can be found inside your members are when you have some live trading account opened through us.

Trade directions

  • 0 - Long & shorts
  • 1 - Longs
  • 2 - Shorts

Indexes

  • ai50
